# VaptVupt - Landing Page

Landing page desenvolvida para o cliente **VaptVupt**, utilizando **Next.js**, e **TailwindCSS**. Projeto totalmente responsivo e pronto para produção.

---

## Tecnologias

- **Next.js** - Framework React para aplicações web.
- **TailwindCSS** - Framework CSS utilitário.
- **TypeScript** - Superset do JavaScript com tipagem.
- **React Hook Form** - Gerenciamento de formulários.
- **Lucide React** - Biblioteca de ícones.
- **EmailJS** - Envio de emails pelo frontend.
- **Tw Animate CSS** - Animações CSS integradas ao Tailwind.

---
